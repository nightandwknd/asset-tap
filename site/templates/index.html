Asset Tap

Generate 3D models from text prompts

Download   Get Started

How It Works

Text → AI Image → 3D Model (GLB) → FBX Export

  1. Enter a text prompt (e.g., "a steampunk robot knight")
  2. AI generates a high-quality image from your prompt
  3. A second AI model converts the image into a 3D model
  4. Optionally export to FBX via Blender for game engines

Features

  • Built-in 3D Viewer — Preview and inspect models before export
  • Multiple AI Models — Choose the best text-to-image and image-to-3D models for your workflow
  • Template System — Create and reuse prompt templates
  • FBX Export — Automatic conversion via Blender
  • Library Management — Browse and organize generated models
  • CLI + GUI — Full-featured command-line and graphical interfaces
  • Zero-Cost Testing — Built-in mock mode for development

Cross-Platform

Official releases for macOS, Windows, and Linux.

  • macOS — Universal DMG with GUI + CLI (Intel + Apple Silicon)
  • Windows — NSIS installer with GUI
  • Linux — .deb package (GUI + CLI) and AppImage
