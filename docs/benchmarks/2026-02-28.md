## Notes

- Generation times include queue wait time, which varies with fal.ai server load. Times are not purely model inference — they reflect real-world conditions including cold starts and queue position.
- TRELLIS 2 and FLUX.2 Dev runs had notable queue wait times (~2 minutes in queue before processing started).
- Hunyuan3D Pro produced the largest GLB output (37 MB) while Meshy v6 produced the most compact (4.0 MB).
- FLUX.2 Pro produced a notably smaller image (203 KB) compared to other models (~1.2-1.7 MB).
- FBX conversion via Blender succeeded for all 3 tested 3D model outputs.
- All runs used the release binary with release-mode configuration paths (`~/Library/Application Support/asset-tap/`).
